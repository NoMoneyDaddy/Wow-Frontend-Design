## Controlled comparison

For every strong/weak or provider comparison, keep identical:

- brief, source project, allowed files/tools/network and context lane;
- skill revision, loaded references, time/token/cost budget and retry policy;
- viewports, browsers, locales, states, input paths and evidence policy;
- hidden functional assertions and anonymized screenshot set.

Score independently on task representation, brand-source calibration, interaction/state completeness, non-landing anti-template behavior, mobile transformation, localization, accessibility, resilience and authored craft. The implementation model never totals its own score. Static anti-pattern signals are alerts only; browser outcomes plus blind human review decide behavior and visual judgment.

Report sample size, ties, disagreement, infrastructure failures, unverified checks and all repairs. Do not publish a cross-model ranking from one site, one judge, different budgets or different tool access.
